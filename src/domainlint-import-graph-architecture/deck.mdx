## **Three Simple Rules**

---

layout: side

---

### **Rule 1: No Deep Imports**

Always import from barrel files (`index.ts`)

<Side>

<Appear>

```typescript
// ❌ Bad
import { Button } from "../features/ui/components/Button";

// ✅ Good
import { Button } from "../features/ui";
```

</Appear>

<Appear>

Enforces clear **public APIs**

</Appear>

</Side>

---

layout: side

---

### **Rule 2: No Cyclic Dependencies**

Between features OR within features

<Side>

<Appear>

```typescript
// ❌ Bad
// authentication/index.ts
import { Dashboard } from "../dashboard";

// dashboard/index.ts
import { useAuth } from "../authentication";
```

</Appear>

<Appear>

```typescript
// ✅ Good
// shared/auth/index.ts
export const useAuth = () => { ... }

// Both features import from shared
```

</Appear>

</Side>

---

layout: side

---

### **Rule 3: Features Are Isolated**

No imports outside of features from inside features

<Side>

<Appear>

```typescript
// ❌ Bad - feature importing from feature
// features/dashboard/index.ts
import { UserProfile } from "../user-profile";

// ✅ Good - extract to shared
// shared/types/user.ts
export type User = { ... }

// Both features import from shared
```

</Appear>

</Side>

---

layout: sidedCode

---

### **Setting Up the Rules**

```json
// @step showLines(1-10) highlight(3-9) name("Configuration")
{
  "rules": {
    "no-deep-imports": {
      "enabled": true,
      "allow": ["@/*", "~/*"]
    },
    "no-cyclic-dependencies": true,
    "features-isolation": {
      "featuresPath": "src/features"
    }
  }
}
```
