### Here's what "faster" looked like

```tsx
function ContactForm() {
  const schema = z.object({
    email: z.string().email(),
    message: z.string().min(1),
  });

  const { register, handleSubmit, formState } = useForm({
    resolver: zodResolver(schema),
  });

  const mutation = useMutation({
    mutationFn: submitContact,
    onSuccess: () => setSuccess(true),
  });

  // + useEffect, useState, error handling, toast...

  return (
    <form onSubmit={handleSubmit((d) => mutation.mutate(d))}>
      <input {...register("email")} />
      <textarea {...register("message")} />
      <button type="submit">Send</button>
    </form>
  );
}
```

### **Testing that same form**

```tsx
it("submits the contact form", async () => {
  render(<ContactForm />);

  await userEvent.type(
    screen.getByRole("textbox", { name: "Email" }),
    "alice@example.com",
  );
  await userEvent.type(
    screen.getByRole("textbox", { name: "Message" }),
    "Hello!",
  );
  await userEvent.click(screen.getByRole("button", { name: "Send" }));

  expect(screen.getByText("Thank you!")).toBeInTheDocument();
});

it("shows validation error on empty email", async () => {
  render(<ContactForm />);

  await userEvent.click(screen.getByRole("button", { name: "Send" }));

  expect(screen.getByText("Email is required")).toBeInTheDocument();
});
```

<Side>

<Appear>

`"Email"`, `"Message"`, `"Send"` — **hardcoded** in every test

</Appear>

<Appear>

Rename "Send" to "Submit"? **Every test breaks.**

</Appear>

<Appear>

Code becomes **frozen**. Nobody dares touch the UI.

</Appear>

</Side>

### **Testing: absorb the complexity**

```tsx
it("submits the contact form", async () => {
  const { form, actions } = renderContactForm();

  await actions.fillEmail("alice@example.com");
  await actions.fillMessage("Hello!");
  await actions.submit();

  expect(form.successMessage()).toBeVisible();
});

it("shows validation error on empty email", async () => {
  const { form, actions } = renderContactForm();

  await actions.submit();

  expect(form.emailError()).toBeVisible();
});
```

<Side>

<Appear>

Selectors defined **once**, in `renderContactForm`.

</Appear>

<Appear>

Rename "Send" to "Submit"? **One place** to change.

</Appear>

<Appear>

Tests describe **intent**, not DOM structure.

</Appear>

</Side>

### **domainlint**: make the invisible visible

```typescript
export const mustUseBarrelFileRule: Rule = {
  name: "must-use-barrel-file",
  check({ graph, config, emitViolation }) {
    for (const edge of graph.edges) {
      // ...
      if (!isBarrelFile(edge.toPath)) {
        const expectedBarrelPath = getBarrelPath(edge.toPath);

        emitViolation({
          code: "mustUseBarrelFile",
          message: `Import from feature barrel "${expectedBarrelPath}" instead.`,
        });
      }
    }
  },
};
```

<Side>

<Appear>

Global rules: **cycles** and **cross-feature imports**

<Appear>
